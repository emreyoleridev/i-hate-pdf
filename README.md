# I Hate PDF 🛠️🎬

**100% Secure. 100% Client-Side. 100% Free.**

I Hate PDF is a premium, high-performance web application designed to solve everything you hate about managing PDF documents. Unlike traditional PDF tools, we respect your privacy: **your files never touch any server.** Every single operation is performed locally within your browser using your device's raw power.

![I Hate PDF Preview](/public/og-image.png) *(Note: Ensure you have an OG image or replace this path)*

## 🚀 Why I Hate PDF?

-   **Zero Uploads:** Your sensitive data stays on your machine.
-   **Blazing Fast:** No upload/download queues. Processing is instant.
-   **No Paywalls:** No "pro" versions, no subscriptions, no credit cards required.
-   **No Ads:** A clean, distraction-free environment for professional work.
-   **Works Offline:** Once loaded, most tools work without an internet connection.

## 🛠️ The Ultimate Toolset (22+ Tools)

### Organize & Optimize
-   **Merge PDF:** Combine multiple documents into one.
-   **Split PDF:** Extract specific pages into new files.
-   **Organize PDF:** Rearrange, delete, or add pages visually.
-   **Compress PDF:** Reduce file size (High-fidelity Mock).
-   **Repair PDF:** Rebuild corrupted document headers.
-   **Crop PDF:** Trim margins and adjust page boundaries.

### Edit & Security
-   **Unlock PDF:** Securely remove password protection.
-   **Protect PDF:** High-fidelity UI for document encryption.
-   **Redact PDF:** Permanently black-out sensitive information.
-   **Sign PDF:** Add digital signatures to your documents.
-   **Watermark:** Stamp text or images across every page.
-   **Rotate PDF:** Fix orientation issues in seconds.

### Converters
-   **Scan to PDF:** Use your webcam to scan physical docs into PDFs.
-   **JPG to PDF:** Turn photos into a professional document.
-   **Word/Excel/PPT to PDF:** Standard conversion tools.
-   **PDF to Word/JPG:** Extract content back to editable formats.

## 💻 Tech Stack

-   **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
-   **Logic Engine:** [pdf-lib](https://pdf-lib.js.org/) & [JSZip](https://stuk.github.io/jszip/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Animations:** [Framer Motion](https://www.framer.com/motion/)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Theming:** [Next Themes](https://github.com/pacocoursey/next-themes) (Optimized for Dark Mode)

## 🛠️ Getting Started

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/emreyoleri/i-hate-pdf.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) in your browser.

### Testing

We maintain a 100% pass rate for our core tool components.
```bash
npm test
```

## 🔒 Security Policy
